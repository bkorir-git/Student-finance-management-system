# Student Finance Management System

A comprehensive Flask-based web application for managing student fees, payments, and financial records.

## Features

- **Student Management**: Add, edit, and track students with detailed information
- **Payment Processing**: Record and manage fee payments with receipt generation
- **Fee Structures**: Define flexible fee structures by grade, term, and type
- **Financial Reports**: Generate comprehensive reports with charts and analytics
- **Balance Tracking**: Automatic balance calculations and payment history
- **Multi-user Support**: Role-based access control (Admin, Accountant, Viewer)
- **Receipt Generation**: Professional PDF-ready receipts for all payments

## Requirements

- Python 3.8+
- MySQL 5.7+ or MariaDB 10.3+
- pip (Python package manager)
